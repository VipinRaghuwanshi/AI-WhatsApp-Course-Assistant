# AI WhatsApp Course Assistant 🚀

An AI-powered WhatsApp automation system built using n8n, OpenAI, and Google Sheets.

This AI Agent automatically:

* Handles WhatsApp course inquiries
* Answers FAQs using AI
* Reads course data from Google Sheets
* Stores student enquiries
* Sends automated replies
* Maintains conversation memory

---

# ✨ Features

* 🤖 AI WhatsApp Chatbot
* 📚 Course Information Retrieval
* ❓ FAQ Automation
* 🧠 Conversation Memory
* 📊 Google Sheets Integration
* ⚡ Real-Time Automation
* 📝 Course Enquiry Storage

---

# 🛠️ Tech Stack

* n8n
* OpenAI
* WhatsApp API
* Google Sheets
* Simple Memory

---

# 🧩 Workflow

```text
WhatsApp Trigger
        ↓
     AI Agent
   ↙    ↓    ↘
Memory  FAQ  Courses Sheet
        ↓
Course Enquiry Sheet
        ↓
 WhatsApp Response
```
